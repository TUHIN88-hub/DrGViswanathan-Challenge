# Problem 01 - Binary Tree Inorder Traversal

## Problem Link
https://leetcode.com/problems/binary-tree-inorder-traversal/

## Difficulty
Easy

## Language
Java

## Concepts Used
- Binary Trees
- Inorder Traversal
- Depth First Search (DFS)
- Recursion

## Approach

- Use recursion to perform an inorder traversal.
- Traverse the left subtree.
- Visit the current node and store its value.
- Traverse the right subtree.
- Return the collected traversal list.

## Time Complexity

O(n)

## Space Complexity

O(h)

where `h` is the height of the binary tree.
