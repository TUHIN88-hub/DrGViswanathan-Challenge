# Problems Solved

## Problem 1: 20. Valid Parentheses
**Platform:** LeetCode  
**Difficulty:** Easy  

### Key Concepts:
- Stack
- String Processing
- LIFO Principle
- Character Matching

### Approach:
A stack was used to store opening brackets while traversing the string. For every closing bracket, the top element of the stack was checked for a valid match. If the brackets did not match or the stack was empty, the string was invalid. A valid expression leaves the stack empty after processing all characters.

### Complexity Analysis:
- **Time Complexity:** O(n)
- **Space Complexity:** O(n)

## Problem 2: 121. Best Time to Buy and Sell Stock
**Platform:** LeetCode  
**Difficulty:** Easy  

### Key Concepts:
- Arrays
- Greedy Algorithm
- Single Pass Traversal
- Optimization

### Approach:
The array was traversed once while maintaining the minimum stock price encountered so far. For each day, the possible profit was calculated and the maximum profit was updated whenever a better transaction was found.

### Complexity Analysis:
- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
