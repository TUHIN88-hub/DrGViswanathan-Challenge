Platform: LeetCode
Difficulty: Medium

Key Concepts:

Constant Time Operations
Object-Oriented Programming
Approach:

Implemented a custom MinStack using two stacks:

One stack stores all the elements.
Another stack maintains the minimum elements.

This allows push(), pop(), top(), and getMin() operations to execute in O(1) time.

Complexity Analysis:

Time Complexity: O(1) for all operations
Space Complexity: O(n)
Problem 2: Two Sum

Platform: LeetCode
Difficulty: Easy

Key Concepts:
Arrays
HashMap
Hashing
One-pass traversal
Approach:

Used a HashMap to store the elements and their indices while traversing the array.

For every element:

Calculated the required complement (target - current element).
Checked whether the complement already exists in the HashMap.
Returned the indices when a matching pair was found.

This optimized the solution from a brute-force approach to a single traversal.

Complexity Analysis:

Time Complexity: O(n)
Space Complexity: O(n)
